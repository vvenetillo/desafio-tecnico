# Somatório de Números Divisíveis por 3 ou 5

Este é um simples script em JavaScript que calcula o somatório de todos os números inteiros divisíveis por 3 ou 5 que são inferiores a um número fornecido.
Desafio proposto pela empresa Escribo Inovação para o Aprendizado.

## Como Usar

### Pré-requisitos

- [Node.js](https://nodejs.org/) instalado

### Instalação

1. Clone ou faça o download deste repositório.

```bash
git clone https://github.com/vvenetillo/desafio-tecnico.git

Certifique-se estar dentro do diretório (cd desafio-tecnico)

npm install (instale as dependências)

npm start 'NÚMERO' (Ex: npm start 10)

# Execução em Tempo Real
Se preferir usar o Nodemon para executar o script em tempo real, use o seguinte comando:

npm run dev 'Número' (Ex: npm run dev 10)

# Exemplo
Alguns exemplo de execução:
npm start 10
# Saída esperada: Somatório dos números divisíveis por 3 ou 5 menores que 10 
npm start 11
# Saída esperada: Somatório dos números divisíveis por 3 ou 5 menores que 11
``````

#### Certifique-se em inicializar o programa com os comandos listados acima.
